# Harbor Provider

Summary of what the provider is for, including use cases and links to
app/service documentation.

## Example Usage

```hcl
provider "harbor" {
  url      = "demo.goharbor.io"
  basepath = "/api/v2.0"
  username = "admin"
  password = "Harbor12345"
}
```
